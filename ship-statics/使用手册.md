## 系统简介

### 功能概述

本系统为武汉理工大学船舶与海洋工程学院《船舶静力学》课程设计专门开发，实现船舶静水力计算与稳性校核的全流程自动化。

**核心功能**：

1. **型值表输入与校验** — 支持 Excel 上传和手动输入
2. **静水力曲线计算** — 14 项静水力参数计算
3. **邦戎曲线计算** — 各站横剖面面积和面积矩计算
4. **稳性横截曲线计算** — GZ 曲线计算与绘制
5. **装载工况稳性校核** — 4 种必做工况自动校核
6. **规范自动判定** — 依据《国内航行海船法定检验技术规则》自动判定
7. **Word 说明书生成** — 一键生成完整课程设计说明书

### 技术规范

- **计算依据**: 盛振邦《船舶原理》上册
- **规范标准**: 《国内航行海船法定检验技术规则（2020）》
- **计算方法**: 梯形法数值积分

## 完整使用流程

### 流程图

```
上传型值表 → 校验确认 → 静水力计算 → 曲线生成 → 稳性校核 → 生成说明书
```

### 详细步骤

#### 步骤 1：输入型值表

**方式 A：手动输入**
1. 在"Step 1: 水线面计算"区域
2. 输入垂线间长 LBP（米）
3. 输入吃水 d（米）
4. 输入型值表数据，格式：每行"站号, 半宽"
5. 点击"计算"按钮

**方式 B：上传 Excel**
1. 在"Step 2: 型值表上传"区域
2. 点击"下载型值表模板"按钮
3. 在 Excel 中填写数据
4. 点击"选择文件"上传
5. 点击"上传并校验"按钮

#### 步骤 2：静水力计算

1. 在"Step 3: 全参数计算"区域
2. 设置吃水范围和步长
3. 点击"计算"按钮
4. 查看计算结果表
5. 点击"导出 Excel"保存结果

#### 步骤 3：曲线绘制

1. 在"Step 4: 曲线绘制"区域
2. 点击"绘制静水力曲线"按钮
3. 点击"绘制邦戎曲线"按钮
4. 查看曲线预览
5. 点击"导出所有曲线"保存图片

#### 步骤 4：稳性计算

1. 在"Step 5: 稳性横截曲线"区域
2. 输入重心高度 KG
3. 输入排水量 Δ
4. 点击"计算 GZ 曲线"按钮
5. 查看曲线预览

#### 步骤 5：工况校核

1. 在"Step 6: 装载工况稳性校核"区域
2. 选择工况（满载出港、满载到港、压载出港、压载到港）
3. 查看工况参数
4. 点击"分析当前工况"或"分析全部 4 种工况"
5. 查看浮态计算结果
6. 查看稳性指标
7. 查看规范判定结果
8. 点击"导出全工况 Excel"保存结果

#### 步骤 6：生成说明书

1. 在"Step 7: Word 说明书生成"区域
2. 填写学生信息（班级、姓名、学号、指导教师）
3. 填写船舶信息（船舶名称、类型）
4. 勾选需要插入的图片
5. 点击"生成完整 Word 说明书"按钮
6. 自动下载 Word 文件
7. 打开 Word 文件，右键目录 → 更新域 → 更新整个目录

## 功能详解

### Step 1：水线面计算（手动输入）

**输入参数**：
- **垂线间长 LBP**: 船舶垂线间长度（米）
- **吃水 d**: 计算吃水（米）
- **型值表**: 各站半宽数据

**计算结果**：
- 水线面面积 Aw
- 漂心纵向坐标 xf
- 惯性矩 It、Il
- 横稳心半径 BM
- 纵稳心半径 BML

### Step 2：型值表上传（Excel）

**Excel 格式要求**：
- 第一行为表头
- 第一列：站号（-5 到 5）
- 第二列及之后：各水线半宽数据

**示例格式**：
```
站号  水线0  水线1  水线2  ...
-5    0.00   0.50   1.00   ...
-4    1.85   2.50   3.20   ...
...
```

### Step 3：全参数计算

**计算参数**（14 项）：
1. 水线面面积 Aw（m²）
2. 水线面形心 xf（m）
3. 排水体积 ∇（m³）
4. 排水量 Δ（t）
5. 浮心纵向 xb（m）
6. 浮心垂向 kb（m）
7. 横稳心半径 BM（m）
8. 纵稳心半径 BML（m）
9. 漂心纵向 xf（m）
10. 每厘米吃水吨数 TPC（t/cm）
11. 每厘米纵倾力矩 MTC（t·m/cm）
12. 方形系数 Cb
13. 棱形系数 Cp
14. 水线面系数 Cw

### Step 4：曲线绘制

**静水力曲线**：
- 显示各参数随吃水变化
- 支持高清导出（PNG、SVG）
- A3 尺寸，300 DPI

**邦戎曲线**：
- 显示各站横剖面面积和面积矩
- 支持高清导出

### Step 5：稳性横截曲线

**计算方法**：变排水量法

**输出参数**：
- 初稳性高度 GM
- 最大复原力臂 GZ_max
- 最大复原力臂角度 θ_max_gz
- 稳性消失角 θ_vanish

### Step 6：装载工况校核

**4 种必做工况**：
1. **满载出港** — 货舱 100%，燃油 100%，压载舱空
2. **满载到港** — 货舱 100%，燃油 10%，压载舱空
3. **压载出港** — 货舱空，压载舱 100%，燃油 100%
4. **压载到港** — 压载舱 85%，燃油 10%

**衡准标准**（《国内航行海船法定检验技术规则》）：
- GM ≥ 0.15m
- GZ_max ≥ 0.20m
- θ_max_gz ≥ 25°
- θ_vanish ≥ 55°
- K ≥ 1.0

### Step 7：Word 说明书生成

**章节结构**：
1. 封面
2. 目录
3. 设计任务书
4. 主尺度确定与型线设计说明
5. 静水力计算
6. 邦戎曲线计算
7. 稳性计算与校核
8. 设计总结
9. 参考文献

**格式规范**：
- 字体：正文宋体 12pt，标题黑体
- 段落：1.5 倍行距，首行缩进 2 字符
- 图表：自动编号

## 常见问题解决

### Q1：启动失败，提示"未检测到 Python 环境"

**解决方法**：
1. 下载并安装 Python 3.8+
   - 下载地址：https://www.python.org/downloads/
   - 安装时**必须勾选** "Add Python to PATH"
2. 重新打开命令提示符，输入 `python --version` 验证
3. 重新运行 `start.bat`

### Q2：依赖安装失败

**解决方法**：
1. 检查网络连接
2. 手动安装依赖：
   ```bash
   pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
   ```
3. 如果仍然失败，尝试使用其他镜像源：
   ```bash
   pip install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/
   ```

### Q3：端口 5000 被占用

**解决方法**：
1. 查看占用端口的程序：
   ```bash
   netstat -ano | findstr :5000
   ```
2. 结束占用进程：
   ```bash
   taskkill /F /PID <进程ID>
   ```
3. 或修改 `app.py` 最后一行，改用其他端口：
   ```python
   app.run(debug=True, port=5001, use_reloader=False)
   ```

### Q4：型值表上传失败

**解决方法**：
1. 确保 Excel 格式正确
2. 第一行必须为表头
3. 第一列为站号，第二列及之后为各水线半宽
4. 不要有合并单元格
5. 不要有空行或空列

### Q5：计算结果显示"NaN"或错误

**解决方法**：
1. 检查型值表数据是否正确
2. 确保半宽值为非负数
3. 确保吃水值为正数
4. 检查 LBP 是否填写正确

### Q6：图片无法插入 Word 说明书

**解决方法**：
1. 确保在 Step 4-6 中生成了相应的曲线图
2. 确保导出了曲线图片
3. 检查图片文件是否存在（在 `outputs` 目录中）

### Q7：Word 目录显示不正确

**解决方法**：
1. 打开 Word 文件
2. 右键点击目录区域
3. 选择"更新域"
4. 选择"更新整个目录"
5. 点击"确定"
