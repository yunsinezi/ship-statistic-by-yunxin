## 📦 项目打包说明

### 打包前准备

在打包项目之前，请确保：

1. **清理临时文件**
   ```bash
   # 删除 Python 缓存
   find . -type d -name "__pycache__" -exec rm -rf {} +
   find . -type f -name "*.pyc" -delete
   
   # 删除临时输出文件
   rm -rf outputs/*
   rm -rf uploads/*
   ```

2. **检查文件完整性**
   - 所有 `.py` 文件存在
   - `templates/index.html` 存在
   - `requirements.txt` 存在
   - `start.bat` 存在

### 打包步骤

#### 方法一：ZIP 压缩包（推荐）

1. **选中项目文件夹** `ship-statics`

2. **右键 → 发送到 → 压缩(zipped)文件夹**

3. **重命名为**：
   ```
   船舶静力学课程设计系统_v1.0_20260326.zip
   ```

4. **压缩包大小**：约 5-10 MB

#### 方法二：使用打包工具

1. **安装打包工具**
   ```bash
   pip install pyinstaller
   ```

2. **创建可执行文件**
   ```bash
   pyinstaller --onefile --windowed app.py
   ```

3. **打包后的可执行文件** 在 `dist` 目录中

## 💻 在其他电脑上运行

### 系统要求

- **操作系统**: Windows 10/11
- **Python**: 3.8 或更高版本
- **浏览器**: Chrome/Edge/Firefox

### 安装步骤

#### 步骤 1：安装 Python

1. **下载 Python**
   - 访问 https://www.python.org/downloads/
   - 下载 Python 3.8+ 版本（推荐 3.10 或 3.11）

2. **安装 Python**
   - 双击安装包
   - **勾选 "Add Python to PATH"**（重要！）
   - 点击 "Install Now"

3. **验证安装**
   ```bash
   python --version
   ```
   应显示 Python 版本号

#### 步骤 2：解压项目

1. 解压 `船舶静力学课程设计系统_v1.0_20260326.zip`
2. 解压到任意目录（建议路径不含中文和空格）

#### 步骤 3：启动系统

1. 进入解压后的目录
2. 双击 `start.bat`
3. 等待自动安装依赖
4. 浏览器自动打开系统界面

## ⚠️ 常见问题

### Q1：同学电脑没有 Python 怎么办？

**解决方法**：
1. 提供 Python 安装包（python-3.x.x.exe）
2. 或者使用 PyInstaller 打包成 `.exe` 文件
   ```bash
   pip install pyinstaller
   pyinstaller --onefile --windowed app.py
   ```

### Q2：打包后体积太大怎么办？

**解决方法**：
1. 删除不必要的文件（如 `.pyc`、`__pycache__`）
2. 不包含 `outputs` 和 `uploads` 目录
3. 使用 7-Zip 等高压缩率工具

### Q3：同学安装依赖失败怎么办？

**解决方法**：
1. 提供离线依赖包
   ```bash
   pip download -r requirements.txt -d ./packages
   ```
2. 同学离线安装：
   ```bash
   pip install --no-index --find-links=./packages -r requirements.txt
   ```

### Q4：路径包含中文导致启动失败？

**解决方法**：
1. 将项目移动到纯英文路径
2. 如 `C:\ship-statics\`
